Cómo usar FTP

Instalación FileZilla

Descarga de la versión gratuita para usuarios.

A la izquierda se nos muestra el ordenador local y a la derecha el servidor cuando nos conectemos.

Alojamiento de la web

Activamos el protocolo FTP y obtenemos los datos de 000webhost para poder subir los ficheros desde FileZilla usando el protocolo FTP.

Enlace con FileZilla

Configuración con los datos proporcionados para enlazarlos con la aplicación FTP.

Hemos configurado Filezilla con los datos que nos ha proporcionado el servidor de alojamiento.

Acceso al servidor gratuito de alojamiento

Acceso al servidor de alojamiento de webs gratuito desde protocolo FTP con FileZilla. En el log se muestra que la conexión ha sido satisfactoria. Estamos conectados al servidor.

Pueden observarse a la derecha los ficheros originales contenidos por la web por defecto creada, y a la izquierda el ordenador local desde el que accedemos.

Subida de los archivos al servidor

Procedemos a la subida de las carpetas que contienen la web al servidor de alojamiento gratuito.

Simplemente arrastramos los archivos que deseamos subir al servidor o directamente las carpetas donde tenemos los ficheros html y css con nuestra web.
